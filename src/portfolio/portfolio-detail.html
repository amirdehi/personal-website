E-Commerce Platform

A modern, responsive e-commerce site focused on user experience.

About The Project

This project was developed to provide a seamless and fast shopping experience for users on any device. The primary goal was to create a mobile-first design with an intuitive interface, from browsing products to the final checkout. The website is powered by a headless CMS, which allows for easy content updates without needing to redeploy the entire application.

I was responsible for the entire front-end development, including component design, state management, and API integration. A key challenge was optimizing performance to ensure fast load times, which was achieved through code splitting and lazy loading of images.

Technologies Used

  • Vue.js
  • Vuex
  • Node.js
  • Stripe API
  • Contentful (CMS)
  • Netlify
